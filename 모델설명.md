# 주상골 골절 X-ray 자동 판독 AI — 모델 설명서

> 작성일: 2026-05-17

---

## 1. 모델 목적

손목 X-ray 영상에서 **주상골(Scaphoid) 골절 여부를 자동으로 판독**하는 딥러닝 앙상블 모델.

주상골 골절은 X-ray에서 골절선이 매우 미세해 초기 진단을 놓치기 쉽고, 치료가 늦어지면 무혈성 괴사(Avascular Necrosis)로 악화될 수 있어 임상적으로 조기 발견이 매우 중요하다.

---

## 2. 모델 구조

### 2.1 앙상블 아키텍처

3개 모델을 **가중 앙상블(2:1:2)** 후 환자·측면 단위로 MAX 집계하여 최종 판정한다.

```
입력 X-ray 이미지
        │
   ┌────┴────────────────────┐
   │                         │
YOLOv8m-cls           timm CNN 계열
(MURA 재학습)    ┌──────────────────────┐
  weight=2      │ DenseNet-121  weight=1│
                │ EfficientNet-B3 w=2  │
                └──────────────────────┘
        │
  가중합 (2:1:2) ÷ 5
        │
  환자·측면 단위 MAX 집계
        │
  임계값 0.652 적용
        │
  골절(Fracture) / 정상(Normal)
```

### 2.2 개별 모델 상세

| 모델 | 파라미터 | 입력 크기 | 역할 |
|------|:--------:|:--------:|------|
| YOLOv8m-cls | 17M | 640px | 정상 판별 특화 (MURA 재학습으로 특이도 개선) |
| DenseNet-121 | 8M | 224px | 골절 민감도 특화 (Dense Connection이 미세 골절선 보존) |
| EfficientNet-B3 | 12M | 300px | 민감도·특이도 균형 (앙상블 핵심 역할) |

### 2.3 가중치 설정 근거

| 모델 | 가중치 | 근거 |
|------|:------:|------|
| YOLOv8m-cls | 2 | MURA 재학습 후 정상 평균 점수 0.503 → 0.122로 가장 큰 개선 |
| DenseNet-121 | 1 | 민감도 94.3%로 최고이나 특이도 55%로 낮아 낮은 가중치 부여 |
| EfficientNet-B3 | 2 | 민감도 88.6% / 특이도 70.0% / AUC 0.850 — 균형 최고 |

### 2.4 임계값 전략

- **기본값:** 0.652 (Val 세트에서 민감도 ≥ 98% 조건 중 특이도 최대인 값)
- **환자 단위 집계:** 동일 환자·동일 방향의 여러 이미지 중 **최대 확률(MAX)** 을 케이스 점수로 사용

## 3. 학습 데이터

### 3.1 원본 데이터 (단일 기관 병원 데이터)

| 분할 | 골절 | 정상 | 합계 |
|------|:----:|:----:|:----:|
| Train | 223장 | 172장 | 395장 |
| Val | 25장 | 22장 | 47장 |
| Test | 35장 | 20장 | **55장** |
| **합계** | **283장** | **214장** | **497장** |

- 총 **76명의 환자** / 환자 단위 분할로 데이터 누수 방지
- **정상 레이블 = 골절 환자의 반대 손** (실제 건강인 X-ray 아님)

### 3.2 외부 데이터 추가 (MURA v1.1)

정상 데이터의 구조적 한계(골절 환자 반대손만 존재)를 해결하기 위해 Stanford MURA v1.1 데이터를 추가.

| 추가 내용 | 수량 | 조건 |
|----------|:----:|------|
| XR_WRIST negative → Train/normal | +350장 | 해상도 ≥150px, study당 최대 2장 |
| XR_WRIST negative → Val/normal | +60장 | study당 최대 1장 |
| Test 세트 | 추가 없음 | 오염 방지 |

**효과:** YOLO 모델의 정상 이미지 평균 골절 확률 0.503 → 0.122

## 4. 학습 방법

### 4.1 timm 모델 (DenseNet-121, EfficientNet-B3)

| 기법 | 설정 |
|------|------|
| 손실 함수 | SmoothedFocalLoss (gamma=2.0, smoothing=0.05) |
| 클래스 균형 | WeightedRandomSampler (1:1 골절:정상) |
| Phase 1 | 15 에포크, backbone 동결 → 분류 헤드만 학습 |
| Phase 2 | 최대 85 에포크, 전체 가중치 해동 (backbone lr=3e-5) |
| Dropout | 0.4 |
| 체크포인트 기준 | 민감도 ≥ 95% 조건 충족 시 0.5×AUC + 0.5×특이도 최대화 |

### 4.2 YOLOv8m-cls

- 기존 best.pt에서 파인튜닝 (Transfer Learning)
- lr=1e-4, AdamW, AMP, dropout=0.3
- 46 에포크 종료, top1_acc=0.897

### 4.3 TTA (Test-Time Augmentation)

추론 시 6~8개 설정 평균:

```
(hflip=False, angle=0°), (hflip=True, angle=0°),
(hflip=False, ±10°),     (hflip=True, ±10°),
(hflip=False, +20°),     (hflip=True, -20°)
```

### 4.4 데이터 증강

| 기법 | 설정 |
|------|------|
| 좌우 반전 | 50% |
| 회전 | ±15° |
| 스케일 | ±40% |
| 이동 | ±10% |
| CLAHE | 실험 결과 **제외** (성능 저하: 81.8% → 65.5%) |
